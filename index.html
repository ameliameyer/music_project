My Liked Songs on Spotify

Data Preprocessing, EDA, Visualizations, Spotify Wrapped, & Music Recommendations

Github Repository

Background

Spotify is an audio-streaming platform with over 80 million tracks which first launched in Sweden in 2008. The platform's content mainly includes music, but extends to podcasts and videos as well from creators around the world. Users can choose a premium paid, ad-free experience with offline listening or the free version which does include ads. Spotify functionality includes recommended playlists and user-created playlists.

About This Project

When users 'heart' a song, they are signifying that they like it, and it's added to what Spotify refers to as a Liked Songs playlist. Songs can also be added to user-created playlists. These can be disclosed as private - limited to the user who created it - or public. Songs can also be added to a playlist by multiple users. These are referred to as collaborative playlists.
In this project, I extracted my Liked Songs from Spotify using Spotipy, a Python library for the Spotify API, in order to perform analysis on my Spotify data. Spotipy allows you to extract the general information about the songs such as the names and artists, as well as the particular audio features such as energy level or danceability.
I also extracted my 2022 Spotify Wrapped Dataset. You can read more about these datasets on the Data Prep page of this project.


In this project, I analyzed the popularity of the tracks, the correlation of the audio features and the change in data attributes over time created visualization of the data using Python and Seaborn. ...

I used the data to answer the following:
- Is there a relationship between different audio features of a song?
- Are the songs I listen to more or less popular?
- Which artists to add the most?
- Is there a pattern to when I add the most songs or a certain type of songs? - seasons, months, days of the week, etc
- Do I add more songs in the beginning or at the end of the month?
- Which artists/songs
- How do the songs I've added compare to the most popular songs on Spotify in 2022, 2021, ...?
