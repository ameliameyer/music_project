Dependencies

  • os - a module that includes function for interacting with the operating system(os)
  • spotipy - a Python library for the Spotify Web API
  • SpotifyClientCredentials - load spotify credentials
  • SpotifyOAuth -
  • load_dotenv -
  • numpy - a Python library used for working with arrays
  • pandas - a Python library for data manipulation and analysis.
  • seaborn - a Python visualization library based on matplotlib used for visualization. It provides a high-level interface for drawing statistical graphics.
  • plotly.express - a Python library that contains functions that can create entire figures at once
  • matplotlib.pyplot - a Python library for creating static, animated, and interactive visualizations in Python
  • KMeans - an unsupervised learning method for clustering data points. It divides data points into K clusters by minimizing the variance in each cluster.
  • StandardScaler - a Python library that contains a function to standardize the data values into a standard format
  • Pipeline - a Python utility for machine learning operations. It allows a linear series of data transforms to be linked together, resulting in a measurable modeling process.
  • TSNE - T-distributed neighbor embedding (t-SNE) is a dimensionality reduction technique that helps users visualize high-dimensional data sets by taking the original data that is entered into the algorithm and matches both distributions to determine how to best represent this data using fewer dimensions.
  • PCA - Principal component analysis (PCA) in Python can be used to speed up model training or for data visualization. This reduces the number of variables in our model
  • euclidean_distances - calculates the distance between two points
  • cdist - calculate the pair-wise distances between the vectors in X
  • warning - allows control of warning messages
  • FeatureCorrelation - calculates Pearson correlation coefficients and mutual information between features and the dependent variable
